## Project Overview

**Nexter**is a fictional luxury real-estate platform landing page.

The project focuses heavily on advanced layout techniques, including:

- Complex multi-column & full-screen grid layouts

- Reusable SCSS partials

- Responsive typography & images

- Modern CSS design patterns

This project allowed me to practice writing clean, scalable, and maintainable CSS code.

## Tech Stack

- HTML5
- CSS3 (Grid & Flexbox)
- Sass (SCSS)
- Responsive Design
- BEM Methodology for CSS

## Key Features

- **Master CSS Grid**: layout complex sections like hero, features, and listings with Grid.
- Fully responsive.
- Modular SCSS structure for scalable and maintainable styles.
